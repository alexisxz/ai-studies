## App to build

## The 5 Steps of a good prompt
- Goal: What you want to create
- Input: What the users provide
- Layout: How the apps should look
- Features: What special features to include
- Output: What the app should output
`You do not put all the blocks and the order does not matter, it is just a way to know the pieces you can provide if something is missing`
