# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository purpose

This is a personal AI/ML learning repository containing study notes and hands-on mini-projects from deeplearning.ai courses.

## Structure

- `deeplearning.ai/ai-for-everyone/` — Theory notes (Markdown) on AI foundations, ML workflows, project strategy, org design, and ethics. PDFs of course slides are in `sliders/`.
- `deeplearning.ai/generative-ai-for-everyone/` — Notes on LLMs, RAG, RLHF, generative AI project lifecycle, and business impact.
- `deeplearning.ai/build-with-andrew/` — Standalone HTML/CSS/JS mini-projects (no build step, open directly in browser).
- `deeplearning.ai/claude-code/` — Notes on Claude Code itself as a study topic.

## Running projects

The `build-with-andrew/` projects are single-file HTML apps with no dependencies or build step:

```bash
open deeplearning.ai/build-with-andrew/<file>.html
```

## Notes format

Course notes live in `README.md` files within each course directory. The root `README.md` serves as a navigation index with section anchors to key topics within each course README.
